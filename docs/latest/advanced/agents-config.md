### 核心背景
会议围绕**云南省科技厅三医联动项目**展开，项目2024年启动、2026年需验收，核心聚焦“五高人群慢病管理”，目标呈现1000万经济效益，参与方包括医院（多科室）、上海交大云南研究院、健之佳、数据治理服务商等。

### 核心讨论内容
1. **经济效益1000万呈现方向**：
- 医疗端：对比DRG分组历史数据，看慢病相关组的总费用、用药/检查费用、平均住院日下降情况；减少重复检查、过度用药的医保/医院成本浪费。
- 患者端：通过互联网医院远程复诊、慢病续方（药品邮寄），节省患者交通、住宿、时间成本；扩大样本测算基层患者相关支出减少额度。
- 运营端：AI辅助问诊缩短诊疗时长、提升医生接诊量，信息化随访替代人工减少医护工作量，数据治理优化病案质量提升效率。

2. **数据治理相关**：
- 核心需求：整合医院PACS、EMR等系统散落的慢病数据，形成标准化数据集，支撑数据互通与效果对比；需动态反馈数据质量问题（如漏填身高体重），推动临床改进。
- 后续计划：先迁移治理历史停用数据，再纳入实时数据，后续可支持科研分析、AI病案编码/质控等拓展应用。

3. **互联网医院应用**：
- 核心功能：慢病健康档案建立、风险分层、线上随访/科普、远程复诊、续方邮寄、可穿戴设备数据监测与远程指导。
- 医生排班：支持固定排班（全职线上）或自主排班（碎片化时间接诊），保障合规与医疗风险可控。

4. **其他关键议题**：
- 检验检查结果互认：参考楚雄模式，打通医院系统避免重复检查，大理州暂未落地；
- 分级诊疗与处方流转：计划联动医共体、药店实现慢病处方线上审核、药品配送，解决基层患者就医不便问题；
- 项目推进：一季度深化场景应用设计，二季度开发，年中部署第二阶段成果；需尽快推广系统使用，完成1万用户量指标，补充人员信息与端口配置。
